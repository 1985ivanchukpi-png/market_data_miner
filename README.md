# Marketplace Data Miner
Професійний інструмент для збору та структурування даних з маркетплейсів (Amazon, eBay, Rozetka).

## Технології:
- **Python** (Requests, BeautifulSoup)
- **Обхід захисту:** Ротація User-Agent та імітація поведінки користувача.
- **Docker:** Повна контейнеризація для запуску на сервері.

## Статус:
В розробці (структура проекту готова).
